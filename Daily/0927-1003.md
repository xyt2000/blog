### ubuntu安装java环境

[Java Software | Oracle](https://www.oracle.com/java/)

下载后解压

```shell
tar -xvf jdk-7u72-linux-i586.gz
```

配置环境变量

```shell
vi /etc/profile
#在末尾行添加环境变量 
JAVA_HOME=/usr/local/java/jdk1.7.0_72
CLASSPATH=.:$JAVA_HOME/lib.tools.jar
PATH=$JAVA_HOME/bin:$PATH
export JAVA_HOME CLASSPATH PATH
#生效环境变量
source /etc/profile 
```

检查安装是否成功

```shell
java -version
```
